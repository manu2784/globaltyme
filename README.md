# GlobalTyme

GlobalTyme is a professional platform designed to streamline global scheduling and time-zone management. This repository contains the source code for the web application, built with a focus on performance, scalability, and a seamless user experience.

**Live Demo:** [https://www.globaltyme.com/](https://www.globaltyme.com/)

---

## 🚀 Features

* **Real-time Timezone Conversion:** Easily sync schedules across different global regions.
* **Responsive Design:** Optimized for desktop, tablet, and mobile viewing.
* **Intuitive UI:** Clean and modern interface for quick navigation.
* **Interactive Scheduling:** Tools built to handle complex international coordination.

## 💻 Getting Started

### Prerequisites

* Node.js (v18+)
* npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone git@github.com:manu2784/globaltyme.git
    cd globaltyme
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
